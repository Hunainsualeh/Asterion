## UI/UX standards
Design for the user, not the default:
- **User-centered** — build around real tasks; make the primary action obvious; no dead ends.
- **Consistent & clear** — one coherent layout, spacing, and interaction language; clear visual hierarchy.
- **Accessible** — sufficient contrast, visible focus, full keyboard use, semantic HTML + ARIA, respect reduced-motion.
- **Real states** — implement empty, loading, error, and success states; interactions must actually work.
- **Design system** — one coherent palette, type scale, spacing, radii, shadows — not ad-hoc values.
- **Distinctive & responsive** — a modern, intentional look with real polish; reflows cleanly across breakpoints.
